# Security Policy

`paint-to-print-3d` is a local conversion library and CLI. It should not require secrets for normal use.

## Reporting A Vulnerability

If you find a vulnerability, please open a GitHub security advisory or contact the maintainer privately through GitHub. Avoid posting working exploit details in a public issue before there is time to review.

## Sensitive Files

Please do not attach:

- API keys or access tokens
- private model files you cannot redistribute
- paid marketplace assets
- customer or business data

For bugs, a small synthetic repro or a public sample model is preferred.
